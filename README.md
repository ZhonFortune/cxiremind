## 项目简介 :page_with_curl:

本项目为超星学习通的通知提醒应用,通过获取超星学习通的通知列表,将通知以日历的形式展示在手机日历中,方便用户查看通知.当前仅支持IPHONE日历,您可将其部署于服务器作为订阅日历使用.
